#Lista de erros da biblioteca

1. (ERRO 1) Dimensão incompatível.
Significa que você pode ter tentado:
1.1. Salvar uma matriz com o número de elementos diferente do declarado;
1.2. Fazer operações com matrizes de dimensões incompatíveis: 
1.2.1. Determinantes com matrizes não quadradas.
1.2.2. Multiplicações com matrizes de ordem incompatível.
1.2.3. Somas ou subtrações com matrizes de dimensões diferentes.

2. (ERRO 2) Elemento não encontrado.
Siginfica que você poder ter tentado:
1.1. Printar elementos que não existem.
1.2. Obter a subMatrix de elementos que não existem.
